Wordle Pro - Multi-Mode 2026 🧩
O Wordle Pro é uma versão evoluída e altamente customizável do famoso jogo de adivinhação de palavras. Desenvolvido em Python, o projeto utiliza a biblioteca CustomTkinter para entregar uma interface moderna, fluida e com suporte nativo ao modo escuro.

🚀 Funcionalidades
Multi-Modo: Jogue com 1, 2, 3 ou até 4 palavras simultaneamente (estilo Quordle).

Dificuldade Variável: Escolha palavras com 5, 6 ou 7 letras.

Interface Moderna: Design inspirado no Wordle original com estética Dark Mode.

Teclado Integrado: Feedback visual das letras já utilizadas diretamente no teclado da tela.

Duração Adaptativa: O número de tentativas aumenta conforme a quantidade de palavras selecionadas.

🛠️ Tecnologias Utilizadas
Python 3.x

CustomTkinter: Para a interface gráfica (UI) moderna.

Random: Para o sorteio aleatório de palavras.

Tkinter (messagebox): Para alertas e notificações de fim de jogo.

📦 Como Instalar
Clone o repositório:

Bash
git clone https://github.com/seu-usuario/wordle-pro.git
Instale a biblioteca necessária:

Bash
pip install customtkinter
Execute o jogo:

Bash
python main.py
🎮 Como Jogar
Ao iniciar, selecione a quantidade de palavras e o tamanho das letras no menu principal.

Clique em JOGAR AGORA.

Digite suas tentativas usando o teclado físico ou clicando no teclado da tela.

Cores de feedback:

🟩 Verde: Letra correta na posição correta.

🟨 Amarelo: Letra existe na palavra, mas está na posição errada.

⬛ Cinza: Letra não existe na palavra.
